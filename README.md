# ADAMS

All digital implementation of classic analog mixed signal VLSI blocks like
[PLL](https://en.wikipedia.org/wiki/Phase-locked_loop) and
[TRNG](https://en.wikipedia.org/wiki/Hardware_random_number_generator).
